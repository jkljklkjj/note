#### SFT（有监督微调）的数据集格式

对于Decoder-only的结构，它们本质是只能做到续写。那问答也不是续写啊，要理解问题再解决问题。那就把问答对作为一个整体，以后就可以通过续写这个操作来做到回答问题

- instruction / input / output或 chat 格式
- system / user / assistant 多轮消息

{"instruction":"解释过拟合","input":"","output":"过拟合是训练集好、泛化差。"}

#### RM（奖励模型）的数据格式

{"prompt":"如何学NLP？","chosen":"先基础后项目","rejected":"随便看看"}

#### PPO（强化学习）的数据格式

- prompt
- response
- reward
- old_logprobs / values / advantages
