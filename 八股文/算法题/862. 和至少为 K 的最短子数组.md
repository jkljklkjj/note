给定一个数组，找到长度最短的连续子数组，它的和最少为k

思路：首先肯定要用前缀和去快速计算一个区间。那么对于一个位置的前缀和，我们要找到离它最近的且满足条件的前缀和
所有满足条件的左端点都会被弹出来，因为后面的前缀和就算和它满足条件也不可能更短
同时维护队列，队尾到队头递增，因为小的永远不够大的更满足区间和最大的要求，而且弹出的元素只会在现在的更左边。我们追求找到一个小的，右的前缀和

```java
public int shortestSubarray(int[] nums, int k) {
	int n = nums.length;
	// [0,i)的和
	long[] pre = new long[n+1];
	for(int i = 0;i<n;i++){
		pre[i+1] = pre[i]+nums[i];
	}
	int res = n+1;
	Deque<Integer> q = new LinkedList<>();
	for(int i = 0;i<=n;i++){
		long sum = pre[i];
		// 找到区间在这个点的所有可行解
		while(!q.isEmpty() && sum - pre[q.peekFirst()]>=k){
			res = Math.min(res, i-q.pollFirst());
		}
		// 让不满足的出队，队列尾到头递增
		while(!q.isEmpty() && pre[q.peekLast()]>=sum){
			q.pollLast();
		}
		q.addLast(i);
	}
	return res<n+1?res:-1;
}
```
