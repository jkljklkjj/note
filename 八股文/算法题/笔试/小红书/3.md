给定一个数组，对于它的子数组`[l,r]`，它的函数f定义为`[l,l],[l,l+1],...,[l,r]`这些子数组的和的最大值。计算多少个子数组的f刚好为k

思路：
想要快速求一个区间和，那就用前缀和数组。对于一个子数组`[l,r]`，它的函数计算`[l,l],[l,l+1],...,[l,r]`的时候**左端点前缀和就确定了**，只要**找到最大的右端点前缀和**就知道它的f。那么我们就要快速找到一个区间的最大前缀和，用单调栈求**每个位置的前缀和作为最大值的区间**。那么**在这个区间，所有包含这个最大前缀和的子区间的结果**是通过这个前缀和作为右端点计算的
但是我们只找结果是f的，所以不用遍历所有的左端点，而是遍历和这个前缀和计算后结果为k的左端点。**用哈希表存储**，如果这个左端点在这些子区间上面就算数，以这个为左端点，然后加上所有右端点的情况即可

```java
import java.io.*;
import java.util.*;

public class Main{
    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StreamTokenizer st = new StreamTokenizer(br);
        st.nextToken(); int n = (int)st.nval;
        st.nextToken(); int k = (int)st.nval;
        int[] nums = new int[n];
        for(int i = 0;i<n;i++){
            st.nextToken(); nums[i] = (int)st.nval;
        }
        long[] preSum = new long[n+1];
        for(int i = 0;i<n;i++){
            preSum[i+1] = preSum[i]+nums[i];
        }
        Stack<Integer> stk = new Stack<>();
		int[][] window = new int[n][2];
        for(int i = 0;i<n;i++){
            window[i][1] = n-1;
        }
        for(int i = 0;i<n;i++){
            while(!stk.isEmpty() && preSum[i+1]>preSum[stk.peek()+1]){
                int pre = stk.pop();
                window[pre][1] = i-1;
            }
            stk.push(i);
        }
        
        stk = new Stack<>();
        for(int i = n-1;i>=0;i--){
            while(!stk.isEmpty() && preSum[i+1]>preSum[stk.peek()+1]){
                int next = stk.pop();
                window[next][0] = i+1;
            }
            stk.push(i);
        }
        //System.out.println(Arrays.toString(preSum));
        //System.out.println(Arrays.deepToString(window));
        int ans = 0;
		// 用哈希表存每个前缀和出现的位置
        Map<Long, List<Integer>> map = new HashMap<>();
        for (int i = 0; i <= n; i++) {
            map.computeIfAbsent(preSum[i], x -> new ArrayList<>()).add(i);
        }

        for (int i = 0; i < n; i++) {
            int lBound = window[i][0];
            int rBound = window[i][1];
            long target = preSum[i + 1] - k;

            List<Integer> list = map.get(target);
            if (list == null) continue;

            // 统计 list 中在 [lBound, i] 内的个数
            int cnt = 0;
            for (int pos : list) {
                if (pos >= lBound && pos <= i) cnt++;
            }

            // 右端点可以从 i+1 到 rBound+1 任意选
            ans += cnt * (rBound - i + 1);
        }
        System.out.println(ans);
    }
    
}
```
