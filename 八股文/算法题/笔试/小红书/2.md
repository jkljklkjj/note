将一个长n的线段切断成多个，所有切出来的长度必须为a、b或c，并且每个长度为a的线段后面不能有长度c的，有多少种切法

思路：很经典的条件动态规划。用动态规划，计算以这三个结尾的数，长度已有i，有多少种情况。然后处理长度为c的结果的时候，不加上前面长度为a的结果即可

```java
import java.util.*;
import java.io.*;
// 注意类名必须为 Main, 不要有任何 package xxx 信息
public class red2 {
    static final int MOD = 1000_000_007;
    // 还剩下长度i，上一段切了a、b或c（j）的方案数
    public static void main(String[] args) throws Exception{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StreamTokenizer st = new StreamTokenizer(br);
        StringBuilder print = new StringBuilder("以下是答案\n");
        st.nextToken(); int T = (int)st.nval;
        while(T--!=0){
            st.nextToken(); int n = (int)st.nval;
            st.nextToken(); int a = (int)st.nval;
            st.nextToken(); int b = (int)st.nval;
            st.nextToken(); int c = (int)st.nval;
            long[][] dp = new long[n+1][3];
            // dp[i-a.length][0]=dp[i][1]+dp[i][2]
            if(a<=n)dp[a][0] = 1;
            if(b<=n)dp[b][1] = 1;
            if(c<=n)dp[c][2] = 1;
            for(int i = 0;i<=n;i++){
                if(i+a<=n){
                    for(int j = 0;j<=2;j++){
                        dp[i+a][0] = (dp[i+a][0]+dp[i][j])%MOD;
                    }
                }
                if(i+b<=n) {
                    for(int j = 0;j<=2;j++){
                        dp[i+b][1] = (dp[i+b][1]+dp[i][j])%MOD;
                    }
                }
                if(i+c<=n) {
                    for(int j = 1;j<=2;j++){
                        dp[i+c][2] = (dp[i+c][2]+dp[i][j])%MOD;
                    }
                }
            }
            for(int i = 1;i<=n;i++){
                print.append(dp[i][0]+dp[i][1]+dp[i][2]);
                print.append(" ");
            }
            print.append("\n");
        }
        System.out.println(print);
    }
}
```

也能优化，因为对于一个长度为`i-(a+c)`的，只有它能接上非法的段构成`i`，所以减去即可
可以这么想，一个节点i的情况，来自这三个情况的，但是不能来自那个非法情况。
```java
long[] f = new long[n + 1];
f[0] = 1;
for (int i = 1; i <= n; i++) {
    if (i >= a) f[i] = (f[i] + f[i - a]) % MOD;
    if (i >= b) f[i] = (f[i] + f[i - b]) % MOD;
    if (i >= c) f[i] = (f[i] + f[i - c]) % MOD;
    // 减去违规的 a+c 情况
    if (i >= a + c) f[i] = (f[i] - f[i - (a + c)] + MOD) % MOD;
}
System.out.println(f[n]);
```
