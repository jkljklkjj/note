每个课程必须修完它的所有必须课程才能修
是否能修完所有的课程

思路：实际上是一个有向图：必须课程为起点，课程为终点。检测它有没有环，这个场景用拓扑排序就可以了。每次随便找一个入度为0的点，删掉，更新它的所有终点的入度。最后如果有节点入度不为0则证明有环

```java
public boolean canFinish(int numCourses, int[][] prerequisites) {
    int[] inDegree = new int[numCourses];
    List<Integer>[] adjacencyList = new ArrayList[numCourses];
    Arrays.setAll(adjacencyList, i -> new ArrayList<>());
    

    for (int[] dependency : prerequisites) { 
        int course = dependency[0]; 
        int prerequisite = dependency[1]; 
        inDegree[prerequisite]++;
        adjacencyList[course].add(prerequisite);
    }
    
    Queue<Integer> queue = new LinkedList<>();
    for (int i = 0; i < numCourses; i++) {
        if (inDegree[i] == 0) {
            queue.offer(i);
        }
    }
    
    while (!queue.isEmpty()) {
        int currentCourse = queue.poll(); 
        
        for (int nextCourse : adjacencyList[currentCourse]) {
            inDegree[nextCourse]--;
            
            if (inDegree[nextCourse] == 0) {
                queue.offer(nextCourse);
            }
        }
    }
    
    for (int degree : inDegree) {
        if (degree != 0) {
            return false; 
        }
    }
    return true;
}
```
