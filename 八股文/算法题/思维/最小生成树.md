给你一张无向连通图，每条边有权值，找到一张子图，经过所有顶点且是一颗树，要求所有出现的边的权值和最小

#### Prim

每次找到已经生成的树的最小边，如果能新增节点就把这条边加入，并把新点的所有边加入优先队列方便下次找最小边

```java
static class Edge implements Comparable<Edge>{
	int from, to, val;
	public Edge(int from, int to, int val){
		this.from = from;
		this.to = to;
		this.val = val;
	}

	@Override
	public int compareTo(Edge other){
		return this.val-other.val;
	}

	@Override
	public String toString(){
		StringBuilder sb = new StringBuilder();
		sb.append("egde=");
		sb.append(from+",");
		sb.append(to+",");
		sb.append(val);
		return new String(sb);
	}
}

public static void main(String[] args) throws Exception{
	BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
	StreamTokenizer st = new StreamTokenizer(br);
	st.nextToken();int n = (int)st.nval;
	st.nextToken();int m = (int)st.nval;
	boolean[] vis = new boolean[n+1];
	int visCnt = 0;
	List<Edge>[] NodeToEdge = new ArrayList[n+1];
	for(int i = 0;i<=n;i++){
		NodeToEdge[i] = new ArrayList<>();
	}
	PriorityQueue<Edge> edges = new PriorityQueue<>(m);
	int sum = 0;
	for(int i = 1;i<=m;i++){
		st.nextToken();int x = (int)st.nval;
		st.nextToken();int y = (int)st.nval;
		st.nextToken();int z = (int)st.nval;
		NodeToEdge[x].add(new Edge(x, y, z));
		NodeToEdge[y].add(new Edge(y, x, z));
	}
	int start = 1;
	for(Edge edge:NodeToEdge[start]){
		edges.add(edge);
	}
	vis[start] = true;
	visCnt++;
	while(visCnt!=n && !edges.isEmpty()){
		Edge min = edges.poll();
		// System.out.println(min);
		if(!vis[min.to]){
			vis[min.to] = true;
			visCnt++;
			sum += min.val;
			for(Edge edge:NodeToEdge[min.to]){
				edges.add(edge);
			}
		}
	}
	if(visCnt!=n) System.out.println("orz");
	else System.out.println(sum);
}
```

#### Kruscal

将所有边放进最小堆，每次取最小的，如果说加了这个边不会导致子图不是树（通过并查集判断两个点是否在同一子图），就加这条边
悟道：每次思考为什么要引入一个数据结构，怎么维护这个数据。比如说我为了快速获取最小边就引入了最小堆，维护的时候只要把所有边加进去。然后为了快速判断是否在同一子图，就引入并查集，然后在新增边的时候维护并查集。
悟道：每次写东西一旦有思路就肯定能实现。首先是实现思路的能力，其次是实现不出bug的能力。难点在于不调试如何知道出了bug。bug的种类其实很简单，一是遍历出错，二是实现流程出错，三是变量维护出错，最难发现的就是变量维护，你要知道每一步操作相关的变量，它们的耦合关系，如果有漏或者错就可以快速发现

```java
static class UnionFind{
	int n;
	int[] head;

	public UnionFind(int n){
		this.n = n;
		this.head = new int[n+1];
		init();
	}

	public void init(){
		for(int i = 1;i<=n;i++){
			head[i] = i;
		}
	}

	public void merge(int x, int y){
		x = find(x); y = find(y);
		if(x!=y) head[x] = y;
	}

	public boolean check(int x, int y){
		return find(x)==find(y);
	}

	public int find(int x){
		return head[x]==x?x:(head[x]=find(head[x]));
	}
}
```
