有起点和终点，还有移动规则和移动路径，计算路径最小的情况的路径和

#### Floyd

最朴素的算法，但是能计算任意两点的距离。
通过遍历所有中间点，起点和终点计算。然后一步步计算所有起点到终点的距离，这样对于需要经过多个点的起点和终点，也只需要获取起点到终点前一个点作为中间点

```java
int n; // 节点数
int[][] dist = new int[n][n];

// 初始化
for (int i = 0; i < n; i++) {
    Arrays.fill(dist[i], Integer.MAX_VALUE / 2);
    dist[i][i] = 0;
}

// 读入边
for each edge (u, v, w) {
    dist[u][v] = w; // 有向图
    // dist[v][u] = w; // 无向图
}

// Floyd 核心
for (int k = 0; k < n; k++) {      // 中间点
    for (int i = 0; i < n; i++) {  // 起点
        for (int j = 0; j < n; j++) { // 终点
            if (dist[i][k] + dist[k][j] < dist[i][j]) {
                dist[i][j] = dist[i][k] + dist[k][j];
            }
        }
    }
}
```

#### SPFA

bfs的解法，能记录负边权和存在环。每次出队节点，遍历他的所有邻居，如果起点经过这个点到对应终点的距离更小就更新，同时这个终点入队方便接近最终终点
同时一个点可能被多次入队，因为起点到它的最短距离更新了，那么起点经过它到其他点的距离自然也需要更新

```java
int[] dist = new int[n];// 起点到点i的最短距离
Arrays.fill(dist, Integer.MAX_VALUE);
dist[start] = 0;

Queue<Integer> queue = new LinkedList<>();
boolean[] inQueue = new boolean[n];// 是否在队列中
queue.offer(start);
inQueue[start] = true;

int[] count = new int[n]; // 用来判断负环

while (!queue.isEmpty()) {
	// 获取节点
    int u = queue.poll();
    inQueue[u] = false;

	// 遍历节点的所有边
    for (Edge e : graph[u]) {
        int v = e.to, w = e.w;
        // 如果经过这个点的距离更短就更新距离
        if (dist[u] + w < dist[v]) {
            dist[v] = dist[u] + w;
            if (!inQueue[v]) {
	            // 遍历这个点
                queue.offer(v);
                inQueue[v] = true;
                count[v]++;
                if (count[v] > n) {
                    // 存在负环
                    return;
                }
            }
        }
    }
}
```

#### dijkstra

理论上最优的算法，一个点一旦被访问过就不会更改它的距离

```java
int[] dist = new int[n];
Arrays.fill(dist, Integer.MAX_VALUE);
dist[start] = 0;

// 优先队列：按距离排序，距离小的先出队
PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[1] - b[1]);
pq.offer(new int[]{start, 0});

boolean[] visited = new boolean[n]; // 是否已确定最短距离

while (!pq.isEmpty()) {
    int[] cur = pq.poll();
    int u = cur[0];
    int d = cur[1];

    if (visited[u]) continue;      // 已经确定过的点不再处理
    visited[u] = true;

	// 遍历所有终点
    for (Edge e : graph[u]) {
        int v = e.to, w = e.w;
        if (dist[u] + w < dist[v]) {
            dist[v] = dist[u] + w;
            pq.offer(new int[]{v, dist[v]});
        }
    }
}
```
