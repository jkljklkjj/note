适合答案会随着某个参数单调改变的情况

有一个**范围**，每次找到范围的中点，分割成两个部分。通过**判断中点是否满足情况**，从而判断答案在哪个范围，每次都能使得查找的范围小两倍，所以时间复杂度O(logn)

二分有三种写法，下面写一下能通用的全开区间写法
```java
// 这种写法最后返回的肯定是check(mid)=true的
int l = -1, r = nums.length;
while(l+1<r){
	int mid = l+(r-l)/2;//(l+r)/2但是防止溢出
	if(check(mid)){
		// 这里假设mid越大越能满足条件
		l = mid;
	} else {
		r = mid;
	}
}
// 此时l+1=r
return r;
```

#### 本质

自变量$x$和其函数$f(x)$具有单调关系，定义域$(l,r)$。现在要求$f(x)$的大小满足一定条件$check(x)$，现在要找到定义域中的$y$，使得$(l,y)$都不满足，$(y,r)$都满足条件

#### 有序数组查找第一个不小于target的数

```java
int l = -1, r = nums.length;
while(l+1<r){
	int mid = l+(r-l)/2;//(l+r)/2但是防止溢出
	if(check(mid)){
		// 这里假设mid越大越能满足条件
		l = mid;
	} else {
		r = mid;
	}
}
// 此时l+1=r
return r;

boolean check(int idx){
	return nums[idx]>=target;
}
```

**题解**

我们要找到第一个不小于target的数，所以这里数的大小为**函数**。然后我们知道这里数的大小和索引大小是**单调的**，索引就是这个函数的**自变量**。

#### 二分答案

上面只讲到了有序数组是什么，有序数组的本质很简单，函数、单调性和自变量都比较明晰。可是有很多时候我们会遇到其它的情况，这时候我们要找到存在单调性的关联，再套用二分模板

##### 思维训练

###### 一

给你一个整数数组 `nums` 和一个正整数 `threshold`  ，你需要选择一个正整数作为除数，然后将数组里每个数都除以它，并对除法结果求和。

请你找出能够使上述结果小于等于阈值 `threshold` 的除数中 **最小** 的那个。

每个数除以除数后都向上取整，比方说 7/3 = 3 ， 10/2 = 5 。

题目保证一定有解。请指出自变量、函数

答：自变量为这个除数的取值，函数为这些数除后的和。可知除数越小，除后的结果越大

###### 二

给你一个长度为 `n` 的字符串 `s` 和一个整数数组 `order`，其中 `order` 是范围 `[0, n - 1]` 内数字的一个 **排列**。

从时间 `t = 0` 开始，在每个时间点，将字符串 `s` 中下标为 `order[t]` 的字符替换为 `'*'`。

如果 **子字符串** 包含 **至少** 一个 `'*'` ，则认为该子字符串有效。

如果字符串中 **有效子字符串** 的总数大于或等于 `k`，则称该字符串为 **活跃** 字符串。

返回字符串 `s` 变为 **活跃** 状态的最小时间 `t`。如果无法变为活跃状态，返回 -1。

答：自变量为有效的时间t，也就是起作用的$order[0,t]$，函数为有效字符串的数目。可知在基础上星号越多，有效字符串的数目越多
