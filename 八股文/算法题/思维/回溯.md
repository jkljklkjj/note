深度搜索的一个细分

#### 本质

深度搜索不会传递引用而回溯会。修改了对应的引用进入下一层后，下一层退出的时候原来的引用的变量不会自动恢复，需要手写回溯操作来恢复

```java
void dfs(int i, List<Integer> nums){
	System.out.println(i);
	for(int num:nums){
		System.out.println(num);
	}
	System.out.println();
	nums.add(100);
	dfs(i+1,nums);
	System.out.println(i);// i还是原来的
	for(int num:nums){
		System.out.println(num);// 和传入的nums不同了
	}
	System.out.println();
}
```

这里的`i`为拷贝传递，`nums`为引用传递。在深搜后发现`i`并未改变，但是`nums`多了添加的元素

那我们就要手动来去除添加的元素，来实现节点状态的维持
```java
void dfs(int i, List<Integer> nums){
	System.out.println(i);
	for(int num:nums){
		System.out.println(num);
	}
	System.out.println();
	nums.add(100);// 修改操作
	dfs(i+1,nums);
	System.out.println(i);// i还是原来的
	for(int num:nums){
		System.out.println(num);// 和传入的nums不同了
	}
	nums.remove(nums.size()-1); // 加了这一行，也就是修改操作的逆操作
	for(int num:nums){
		System.out.println(num);// 和传入的nums一样了
	}
	System.out.println();
}
```
