前缀和能快速查询区间和

$$
sum[i] = a[0]+a[1]+...+a[i]
$$
$$
a[l]+a[l+1]+...+a[r-1]+a[r] = (a[0]+...+a[r])-(a[0]+...+a[l-1])
= sum[r] - sum[l-1]
$$
```java
int[] nums = new int[]{1,2,3,4,5};
int[] preSum = new int[nums.length+1];
for(int i = 0;i<n;i++){
	preSum[i+1] = preSum[i]+nums[i]; 
}

int sum(int l, int r){
	return preSum[r+1] - preSum[l];
}
```

差分能快速修改区间，但是最后要恢复成原数组
$$
\begin{aligned}
diff[i] &= a[i] - a[i-1] \\
diff[0] &= a[0]
\end{aligned}
$$

$$
restore[i] = diff[0]+...+diff[i]
$$
```java
int[] nums = new int[]{1,2,3,4,5};
int[] diff = new int[nums.length];
diff[0] = nums[0];
for(int i = 1;i<n;i++){
	diff[i] = nums[i]-nums[i-1];
}
update(l,r,val);
int[] restore = new int[nums.length];
restore[0] = diff[0];
for(int i = 1;i<n;i++){
	restore[i] = restore[i-1]+diff[i];
}

void update(int l, int r, int val){
	diff[l] += val;
	diff[r+1] -= val;
}
```

同时还有个支持同时修改和遍历时还原的
```java
int[] add = new int[n+w];
int now = 0;
for(int i = 0;i<n;i++){
	now += add[i];
	now += val;
	// [i,i+w-1]这个w长的区间加上val
	add[i+w] -= val;
}
```
