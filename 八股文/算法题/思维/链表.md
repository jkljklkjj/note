#链表
链表是一个区别于数组的数据结构

#### 本质

每个元素不止有值，还有指向下一元素的指针
不能通过偏移量获取对应元素，只能通过上一元素来获取下一元素

所以链表是指针的艺术
```java
class ListNode{
	int val;
	ListNode next;
	public ListNode(int val){
		this.val = val;
	}
	public ListNode(int val, ListNode next){
		this.val = val;
		this.next = next;
	}
}
```

链表的题目，难度根本在于无法通过偏移量获取元素，所以思考的时候可以集中于如何通过指针实现对应的节点的查找

#### 前后指针

要获取某节点的第前k个节点

先初始化两个指针在头结点，然后后指针前移k步，然后每次两个指针同时移动。当后节点到达这个某节点的时候，前节点即为目标

#### 快慢指针

两个指针以不同的速度移动。当快指针移动到尾的时候，慢指针就移动到中点（偶数长度移动到偏右）

不过注意，如果是进行链表排序，需要将慢指针移动到偏左
