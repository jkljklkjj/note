给定一个包含括号和加减号的表达式字符串，计算结果

思路：和字符串解码类似，暂存左括号左边的计算结果和操作符，遇到左括号就入栈并重置暂存值

悟道：其实栈的相关处理很类似，进入子问题的时候暂存子问题前面的结果，子问题解决后就可以通过前面的结果和子问题的结果解答出问题的结果。同时要知道一个变量的作用是什么，它的生命周期怎么走，遇到什么情况它要怎么处理才能维持它的性质

```java
public int calculate(String s) {
	Stack<Integer> stack = new Stack<>();
	int sign = 1; // 暂存指针左边的符号
	int res = 0; // 结果
	int num = 0; // 解析数字
	
	for (int i = 0; i < s.length(); i++) {
		char c = s.charAt(i);
		if (Character.isDigit(c)) {
			// 解析数字
			num = num * 10 + (c - '0');
		} else if (c == '+') {
			// 处理左边的数字后重置数字
			res += sign * num;
			sign = 1;
			num = 0;
		} else if (c == '-') {
			res += sign * num;
			sign = -1;
			num = 0;
		} else if (c == '(') {
			// 暂存括号左边的结果和运算符
			stack.push(res);
			stack.push(sign);
			res = 0;
			sign = 1;
		} else if (c == ')') {
			res += sign * num; // 目前为括号里面的结果
			num = 0;
			// 左边的结果和括号内的结果运算
			res = stack.pop() * res + stack.pop();
		}
		// 空格忽略
	}
	res += sign * num;
	return res;
}
```
