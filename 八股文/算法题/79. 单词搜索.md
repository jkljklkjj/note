给出一个字符矩阵，查找有没有路径为目标单词

思路：路径上进行深度搜索，注意边界条件。同时搜索过的地方用特殊符号标记作为不可达，再次搜索的时候恢复原符号

```java
int[][] dirs = {{-1,0},{1,0},{0,-1},{0,1}};
public boolean exist(char[][] board, String word) {
	int n = board.length;
	int m = board[0].length;
	boolean tmp = false;
	for(int i = 0;i<n;i++){
		for(int j = 0;j<m;j++){
			if(dfs(board,i,j,word,0)){
				tmp = true;
			}
		}
	}
	return tmp;
}

public boolean dfs(char[][] board, int x, int y, String word, int idx){
	int n = board.length;
	int m = board[0].length;
	if(idx==word.length()) return true;
	if(x<0||y<0||x>=n||y>=m||board[x][y]=='#') return false;
	if(word.charAt(idx)!=board[x][y]) return false;
	char record = board[x][y];
	board[x][y]='#';
	boolean tmp= false;
	for(int[] dir:dirs){
		if(dfs(board,x+dir[0],y+dir[1],word,idx+1))
		tmp = true;
	}
	board[x][y] = record;
	return tmp;
}
```
