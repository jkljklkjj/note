给出括号总数，生成所有合法的括号字符串

思路1：想要给一个合法的字符串合法地添加左括号或右括号，只要满足条件就可以。左括号随便加，只要不超过最大数量；右括号只要不超过已有字符串的左括号数量即可

```java
public List<String> generateParenthesis(int n) {
	List<String> ans = new ArrayList<String>();
	backtrack(ans, new StringBuilder(), 0, 0, n);
	return ans;
}

public void backtrack(List<String> ans, StringBuilder cur, int open, int close, int max) {
	if (cur.length() == max * 2) {
		ans.add(cur.toString());
		return;
	}
	if (open < max) {
		cur.append('(');
		backtrack(ans, cur, open + 1, close, max);
		cur.deleteCharAt(cur.length() - 1);
	}
	if (close < open) {
		cur.append(')');
		backtrack(ans, cur, open, close + 1, max);
		cur.deleteCharAt(cur.length() - 1);
	}
}
```

思路2：可以看到上面的所有操作只有两种，加上左括号或右括号。所以可以设计动态规划状态转移方程，在左括号有`i`个和右括号有`j`个括号的情况下
$$
dp[i][j] = dp[i-1][j](i < n) + dp[i][j-1](j < i)
$$
其实动态规划和利用子树结果的dfs原理差不多，都是求一个状态的时候子状态已经得到答案
```java
public List<String> generateParenthesis(int n) {
	List<StringBuilder>[][] dp = new List[n+1][n+1];
	for(int i = 0;i<=n;i++){
		for(int j = 0;j<=n;j++){
			dp[i][j] = new ArrayList<>();
		}
	}
	dp[0][0].add(new StringBuilder());
	for(int i = 1;i<=n;i++){
		StringBuilder copy = new StringBuilder(dp[i-1][0].get(0));
		dp[i][0].add(copy.append('('));
	}

	for(int i = 1;i<=n;i++){
		for(int j = 1;j<=i;j++){
			for(StringBuilder sb: dp[i-1][j]){
				StringBuilder copy = new StringBuilder(sb);
				dp[i][j].add(copy.append('('));
			}
			for(StringBuilder sb: dp[i][j-1]){
				StringBuilder copy = new StringBuilder(sb);
				dp[i][j].add(copy.append(')'));
			}
		}
	}
	List<String> ans = new ArrayList<>();
	for(StringBuilder sb: dp[n][n]){
		ans.add(new String(sb));
	}
	return ans;
}
```
