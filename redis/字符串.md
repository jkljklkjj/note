设置字符串
```bash
set key value
```
设置多个
```bash
mset [key value...]
```
获取字符串
```bash
get key
```
获取多个（much）
```bash
mget [key...]
```
检测key是否有内容
```bash
exist key
```
获取字符串长度
```bash
strlen key
```
自增减整数类字符串
```bash
incr key
decr key
```
增加减少特定数值
```bash
incrby key value
decrby key value
```
获取value的二进制某位
```bash
getbit key offset
```
设置value的二进制某位
```bash
setbit key offset
```
查看某区间的1的个数
```bash
bitcount key (start end)
```
